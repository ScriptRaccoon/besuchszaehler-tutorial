# Website mit Besuchszähler

Dieses Repository enthält eine SvelteKit-Anwendung, die Besuche zählt und den Gerätetyp jedes Besuchs erfasst. Die Daten werden in einer SQLite-Datenbank gespeichert.

Die Anwendung ist minimalistisch gehalten und eignet sich als Einstieg in datensparsame Tracking-Implementierungen ohne Drittanbieter.

Dieses Projekt ist der Code zu einem kurzen YouTube-Tutorial auf meinem Kanal. Für ausführlichere Varianten siehe:
